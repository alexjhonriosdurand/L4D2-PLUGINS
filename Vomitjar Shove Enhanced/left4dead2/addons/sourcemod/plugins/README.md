## ⚡ Instalación
Sigue estos pasos para instalar el plugin:

1. 📂 Coloca el archivo `.smx` en tu servidor en la ruta:
left 4 dead 2/left4dead2/addons/sourcemod/plugins
2. 🔄 Reinicia el servidor para que el plugin se cargue automáticamente.  

> ⚠️ Asegúrate de tener Metamod y SourceMod actualizados para evitar conflictos.

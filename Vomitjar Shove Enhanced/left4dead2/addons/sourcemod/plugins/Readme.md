# 🦠 [L4D2] Vomitjar Shove Enhanced

**Versión:** 1.3-clean  
**Autor:** Shadow L4D2  

**Descripción:**  
Este plugin para **Left 4 Dead 2** agrega un efecto de "vómito" cuando un jugador con **VomitJar** empuja a otro jugador o a un infectado. Incluye:

- ✨ **Glow temporal** en el jugador afectado.  
- 🌈 **Fade dinámico de pantalla** simulando salpicadura.  
- 🔊 **Efecto sonoro** de splash de vómito.  
- 👥 **Efectos a jugadores cercanos** (radio 150 unidades).  
- 🟢 **Compilación limpia**, listo para instalar como `.smx`.

---

## 📦 Instalación

1. Copia el archivo compilado `.smx` (`vomitjar_shove_enhanced.smx`) a sourcemod/plugins

2. Reinicia el servidor o carga el plugin manualmente:

```bash
sm plugins load vomitjar_shove_enhanced

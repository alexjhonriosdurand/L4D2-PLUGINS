## CVars (si aplica) | 配置变量

*(Si tu plugin tiene variables de configuración que se guardan en un archivo .cfg, lístalas aquí para que los usuarios puedan ajustarlas.)*

Un archivo `smoker_tongue_fix.cfg` (o el nombre de tu archivo) se generará automáticamente en `cfg/sourcemod/` al primer uso. Contendrá las siguientes CVars:

*   `smoker_fix_enable "1"`: Habilita (1) o deshabilita (0) el plugin. (Predeterminado: "1")
*   `smoker_fix_no_shoot "1"`: Si está en 1, los sobrevivientes no pueden disparar mientras son arrastrados. (Predeterminado: "1")
*   `smoker_fix_damage_interval "0.8"`: Intervalo de daño al ser arrastrado (segundos). (Predeterminado: "0.8")
